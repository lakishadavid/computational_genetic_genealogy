About Legacy Labs

These are the original lab notebooks from previous course offerings. While the course has now transitioned to a new format with improved labs (available in the main course sections), these legacy notebooks remain available for reference or for students who started with these materials.

Each lab listed below is directly accessible through Google Colab, allowing you to run these notebooks in a fully-featured Python environment without any local setup. Simply click the "Open in Colab" button for any lab you wish to explore.

These labs cover various aspects of genetic data processing, IBD detection, simulation, and pedigree reconstruction using the Bonsai system.
