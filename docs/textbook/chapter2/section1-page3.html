Why VSCode?

Visual Studio Code (VSCode) is a powerful, free code editor that we'll use throughout this course. We've chosen VSCode because it:

  • Provides excellent integration with Git and Docker
  • Offers a consistent experience across all operating systems
  • Includes built-in features for scientific computing
  • Is widely used in both research and industry

Installation Instructions

Step 1: Download and Install VSCode

Windows

  • Regular Windows: Download and run the .exe installer
  • WSL2 users: Download the .exe installer, but also see the WSL extension setup below

macOS

  • Choose between Intel or Apple Silicon versions
  • Drag Visual Studio Code.app to your Applications folder

Linux

  • Select the appropriate package format (.deb, .rpm, etc.)
  • Use your package manager to install the downloaded package

Essential Extensions

Python Extension Pack

Essential for all Python development in this course

  • Python language support
  • Debugging tools
  • Jupyter notebook integration

Configuration

Basic Settings

Open Settings with Ctrl+, (Windows/Linux) or Cmd+, (macOS)

  • Enable "Auto Save": Files → Auto Save
  • Set "Editor: Font Size": Search for "font size" in settings
  • Enable "Editor: Word Wrap": Search for "word wrap" in settings

Verification Steps

Basic Python Test

  1. Create a new file: File → New File or Ctrl+N/Cmd+N
  2. Save it as test.py
  3. Type: print("Hello, Computational Genetic Genealogy!")
  4. Run: Click play button or right-click → Run Python File
